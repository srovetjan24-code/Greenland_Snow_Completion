[项目名称]：格陵兰岛积雪深度张量补全系统 (Greenland Snow Completion)

1. 项目简介
本项目旨在解决格陵兰岛（Greenland）遥感积雪数据高度稀疏（缺失率通常超过90%）的问题。系统基于改进的 BNLFT 框架，通过引入空间图拉普拉斯约束和时间周期性自回归模型，实现了对大规模、高维度缺失积雪张量的精准恢复。

2. 核心算法与技术实现
- BNLFT 框架：将积雪张量分解为空间因子 (S)、维度因子 (D) 和时间因子 (T)，并引入线性偏置项 (a, b, c) 以捕获系统性偏差。
- 空间图约束 (Graph-based Regularization)：在空间因子更新中加入二阶空间约束，利用相邻格点间的地理相关性，确保补全结果的空间连续性。
- 时间周期自回归 (Periodic AR)：在时间因子更新中引入周期先验参数 (n_d)，利用积雪的季节性演化规律（如 365 天周期）增强序列预测的鲁棒性。
- 滑动窗口分块融合 (Sliding Window Chunking)：针对格陵兰岛海量格点，采用分块处理策略，并配合“线性渐变权重窗口”进行平滑融合，有效消除了块边缘的条纹和“补丁感”。
- 负值兼容优化：区别于传统的非负张量分解，本项目采用标准正态分布初始化，并移除了非负约束，以适应包含负值的积雪异常检测或标准化数据。

3. 目录结构
├── models/
│   ├── lft_model.py      # 定义 BNLFT 模型结构与初始化
│   └── optimizer.py      # 实现带时空约束的随机梯度下降优化器
├── modules/
│   └── transforms.py     # 图拉普拉斯、分数阶微分及周期矩阵生成工具
├── utils/
│   └── math_ops.py       # 提供 Swish 激活函数导数等数学运算
├── main.py               # 主程序：负责数据加载、分块训练及平滑结果生成
└── requirements.txt      # 项目依赖清单

4. 快速开始
- 环境需求：Python 3.10+
- 安装依赖：pip install -r requirements.txt
- 运行补全：直接执行 python main.py。程序将读取指定 CSV 路径数据，并在完成后生成 'Greenland_Snow_Final_Smooth.csv'。

5. 技术特性
- 鲁棒性：自动识别并跳过 NaN 缺失值，仅针对有效观测值进行学习。
- 灵活性：支持动态调整 Rank（潜在因子数量）和 Overlap（重叠区间）以平衡计算速度与精度。
